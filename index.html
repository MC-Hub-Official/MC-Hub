Welcome to MC Hub!

MC Hub is a free website with amazing tips, seeds, facts and command guides to help you become a Minecraft PRO in no time!

Minecraft is a trademark of Microsoft Inc, Mojang Studios AB, or Notch Development AB. This is an un-official fan-made site.


We created MC Hub for three reasons.

We wanted Minecraft players everywhere to learn what we had learnt to make their gaming experience much more enjoyable.

We mainly played Bedrock Edition, and could never find any good Bedrock seeds, so we decided to spend some time creating a Bedrock seed repertoire.

We started with only a tips page and a seeds page, but a while into the project we also got bored of explaining to our nooby friends how to work commands, so we added a commands page.

And to add a bit of humor to the project, we created the facts page, for some interesting fun facts about Minecraft that you may have never known.
